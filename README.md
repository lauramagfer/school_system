# school_system
Projeto desenvolvido em Java como prática de Programação Orientada a Objetos. O sistema permite cadastrar alunos, registrar notas, consultar boletins e visualizar estatísticas da turma.

## Funcionamento
O usuário escolhe uma opção de acordo com aquilo que ele deseja realizar.

## Objetivos do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Programação Orientada a Objetos
- Encapsulamento
- Organização de responsabilidades entre classes
- Manipulação de listas (ArrayList)
- Estruturas de repetição
- Métodos
- Modelagem de entidades

## Estrutura do projeto

- **Main:** Responsável pelo menu e inicialização do sistema.
- **Student:** Representa um aluno(a) com seus atributos e comportamentos.
- **SchoolSystem:** Responsável por gerenciar os alunos e executar as funcionalidades do sistema.

## Aprendizados

Durante o desenvolvimento deste projeto compreendi melhor:

- como distribuir responsabilidades entre classes;
- quando um comportamento pertence ao objeto;
- como trabalhar com coleções de objetos;
- importância da organização do código;
- fundamentos da orientação a objetos.
