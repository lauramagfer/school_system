# school_system
Projeto desenvolvido em Java como prática de Programação Orientada a Objetos. O sistema permite cadastrar alunos, registrar notas, consultar boletins e visualizar estatísticas da turma.

## Funcionamento
O usuário escolhe uma opção de acordo com aquilo que ele deseja realizar.
